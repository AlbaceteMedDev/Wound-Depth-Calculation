# Wound-Depth-Calculation

3-D wound depth measurement formula based on Dr. Shaun Carpenter's True-Graft conformable drape methodology.

## Quick Summary

Traditional wound measurement (L × W × D) overestimates wound volume by **2.55×** compared to actual volume (260 cm³ vs 102 cm³ in Dr. Carpenter's demonstration). This leads to massive graft waste.

**The solution**: Use a conformable surgical drape pressed into the wound bed to capture true 3-D surface area as a 2-D template.

```
Graft Area = A_conform + (Perimeter × 0.5 cm)
```

Where `A_conform` is the flattened drape area and `0.5 cm` is the fixation overlap border.

See [depth-measurement-formula.md](depth-measurement-formula.md) for the full formula derivation, step-by-step clinical procedure, comparison tables, and documentation template.
